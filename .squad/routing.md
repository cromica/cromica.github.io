# Squad Routing

## Default routing

| Work type | Primary owner | Support |
|---|---|---|
| Grammar check, proofreading, style cleanup | Theodora | Mustang |
| Final editorial sign-off | Mustang | Theodora |
| Hugo templates, layouts, theme work | Sevro | Mustang |
| Content promotion, social snippets, brand campaigns | Roque | Dancer |
| Career-positioning review, executive audience fit | Dancer | Roque |
| Cross-cutting roadmap or platform/editorial tradeoffs | Mustang | Sevro, Theodora |

## Heuristics

- If the request names a squad member, route to that person first.
- If the task changes both content and platform behavior, route to Mustang first.
- If the task is publication-readiness, involve Theodora before publishing.
- If the task is reach, audience growth, or personal branding, involve Roque.
- If the task is about leadership narrative, recruiter appeal, or career leverage, involve Dancer.
